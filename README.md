# Flask Calculator

A simple web-based calculator built using Python Flask, HTML, and CSS.
---
## Features
- Add, Subtract, Multiply, Divide two numbers
- Handles decimal and integer inputs
- Displays clear error on division by zero
---
## Technologies Used
- Python
- Flask
- HTML
- CSS
---
